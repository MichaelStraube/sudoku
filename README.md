# Sudoku
A simple sudoku solver.

<img src="share/screenshot.png" alt="sudoku" align="right" height="400">

## Build
```
mkdir build
cd build
cmake ../ -DCMAKE_BUILD_TYPE=Release
make
```
## Install
```
sudo make install
```
